# Blogging API

Second Semester Backend NodeJS Examination Project

## Features

- User signup & login
- JWT Authentication (1 hour expiry)
- Create blog (default draft)
- Publish blog
- Edit blog
- Delete blog
- Owner blog listing
- Public blog listing
- Pagination (default 20 per page)
- Filter by state
- Search by author, title, tags
- Order by read_count, reading_time, timestamp
- read_count increments when blog is viewed
- reading_time calculated automatically
- Full test coverage

## Tech Stack

- Node.js
- Express
- MongoDB
- Mongoose
- JWT
- Jest
- Supertest
